# Powerbi-Superstore-Dashboard

## Overview
Interactive Power BI dashboard with dynamic metric selection (Sales, Profit),
YoY growth analysis, and geographical insights.

## Features
- Dynamic Sales / Profit / Quantity toggle
- YoY growth indicators
- Top 5 States & Segments
- Interactive map & decomposition tree
- Custom tooltips

## Tools
- Power BI
- DAX
- Sample Superstore dataset
